TAKES A WHILE TO LOAD.

ABSTRACT: 
Finding the optimal time to release a movie is a pivotal component to the overall financial success of a movie. The overall research objective is to predict the ideal time for a studio to release their movie. Game Theory and Herfindahl Index are used to develop a model with publicly available data from boxofficemojo.com. Essentially predicting the ideal time for a studio to release a movie can potentially stimulate financial growth for a variety of movie studios.


HOW TO USE GRAPH:
Adjust independent variable to Time from the dropdown button on along the x-axis.
Adjust dependent variable to Gross from the dropdown button on the y-axis.
Select play to watch data change over time.

You can also select a specific movie to watch and track how it performs in comparison with others by selecting the Trials checkbox
